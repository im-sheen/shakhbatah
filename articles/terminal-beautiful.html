إعداد تيرمنال Kitty احترافي في Hyprland

كيف تضبط Kitty مع Zsh وPowerlevel10k للحصول على تيرمنال سريع وجميل — الخطوط والـ blur والـ prompt.

١٦ مايو ٢٠٢٦

الإعداد الأساسي لـ Kitty

# في ~/.config/kitty/kitty.conf

# الخط — استخدم Propo للأيقونات
font_family      GeistMono Nerd Font
font_size        13.0

# إيقاف الصوت المزعج
enable_audio_bell no

# حجم النافذة الافتراضي
remember_window_size  no
initial_window_width  78c
initial_window_height 23c

# المسافة الداخلية
window_padding_width 10

# الـ Blur (يحتاج background_opacity أقل من 1)
background_opacity 0.85

# مؤشر النص
cursor_shape block
cursor_blink_interval 0

# عدم محاذاة النص للـ shell
shell_integration no-cursor

# إيقاف VSYNC
sync_to_monitor no

تفعيل Blur في Hyprland

# في decoration.conf
decoration {
    blur {
        enabled = true
        size = 8
        passes = 2
    }
    active_opacity = 0.9
    inactive_opacity = 0.8
}

تثبيت Zsh وOh My Zsh

# تثبيت Zsh وتعيينه كـ shell افتراضي
sudo pacman -S zsh
chsh -s /bin/zsh

# تثبيت Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

تثبيت Powerlevel10k

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git   ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

في ~/.zshrc:

ZSH_THEME="powerlevel10k/powerlevel10k"

أعد فتح التيرمنال وسيبدأ wizard الإعداد تلقائياً:

p10k configure

إعدادات Powerlevel10k الموصى بها

  • Style → Lean
  • Color depth → 8 colors (يستخدم ألوان Pywal/Matugen)
  • Prompt Lines → Two
  • Frame → Left
  • Enable Transient Prompt → Yes

Ligaturs

إن أردت الـ ligaturs (تحويل -> إلى سهم)، تأكد أن خطك يدعمها. الخطوط مثل GeistMono وFira Code وJetBrains Mono تدعمها.
