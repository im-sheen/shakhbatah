أفضل مدير ملفات لـ Hyprland

مقارنة بين أشهر مديري الملفات — Nautilus وThunar وغيرهما مع ذكر المزايا والعيوب.

Nautilus — الأجمل، لكن فيه عيب

Nautilus مبني على GTK4 — رسومياً هو الأجمل وأكثر المديرين حداثةً. لكن GTK4 لا يدعم Live Theme Reloading، مما يعني أنك ستحتاج لإغلاقه وإعادة فتحه عند تغيير الثيم. 

sudo pacman -S nautilus

إغلاقه أحياناً لا يكفي — يجب قتله كاملاً: 

killall -9 nautilus

Thunar — الأفضل مع Theme Switcher

Thunar مبني على GTK3 — يتغير مظهره فوراً مع كل تغيير للثيم دون الحاجة لإعادة فتحه. الخيار الأمثل إن كنت تستخدم theme switcher ديناميكي. 

sudo pacman -S thunar thunar-archive-plugin

Dolphin — ملك Qt

إن كنت تستخدم تطبيقات KDE كثيراً وأعددت Kvantum بشكل صحيح، Dolphin يتوافق مع مظهر النظام ويقدّم ميزات قوية مثل Split View والـ Tabs المدمج. 

sudo pacman -S dolphin

Yazi — مدير ملفات التيرمنال

إن كنت تفضّل العمل من التيرمنال، Yazi هو الأسرع والأجمل — مع دعم معاينة الصور مباشرة في التيرمنال. 

sudo pacman -S yazi

التوصية

إن كنت تستخدم theme switcher ديناميكي → Thunar. إن كنت تهتم بالجماليات أكثر ولا تغيّر الثيم كثيراً → Nautilus. إن كنت تفضّل الـ terminal → Yazi.
