️ تخصيص المؤشر في Hyprland

كيف تثبّت وتغيّر cursor theme في Hyprland — من gnome-look حتى التطبيق عبر hyprctl.

١٦ مايو ٢٠٢٦

أين تجد الـ Cursors

gnome-look.org → Cursors — أكثر من 1600 cursor theme مجاني.

بعض الخيارات الشائعة في AUR:

# مؤشر Material You (Bibata)
yay -S bibata-cursor-theme

# مؤشر macOS
yay -S apple-cursor

# مؤشر Adwaita الافتراضي (أنيق وبسيط)
sudo pacman -S adwaita-cursors

تثبيت Cursor يدوياً

# نزّل الملف من gnome-look.org
cd ~/Downloads
tar -xzf cursor-theme.tar.gz

# اصنع مجلد icons إن لم يكن موجوداً
mkdir -p ~/.icons

# انقل الثيم
mv cursor-theme-folder ~/.icons/

تطبيق الـ Cursor

الطريقة الأولى: عبر nwg-look (واجهة رسومية)

yay -S nwg-look

افتح nwg-look واختر الـ cursor من قسم Mouse Cursor واضغط Apply.

الطريقة الثانية: عبر hyprctl (الأموثوق)

# تطبيق فوري
hyprctl setcursor CursorThemeName 24

# مثال
hyprctl setcursor macOS 24
hyprctl setcursor Bibata-Modern-Classic 24

جعل التغيير دائماً

أضف في autostart.conf:

exec-once = hyprctl setcursor macOS 24

أو عبر متغيرات البيئة:

env = XCURSOR_THEME,macOS
env = XCURSOR_SIZE,24
⚠️ إن كنت تستخدم theme switcher يغيّر المظهر ديناميكياً، استخدم hyprctl setcursor في سكريبت الثيم بدلاً من متغيرات البيئة الثابتة.

حجم المؤشر الموصى به

  • شاشة 1080p → حجم 24
  • شاشة 1440p أو أعلى → حجم 32 أو 36
  • HiDPI → حجم 48
