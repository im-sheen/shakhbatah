متغيرات البيئة الأساسية في Hyprland

متغيرات البيئة التي لا يخبرك أحد عنها — لإصلاح تطبيقات Electron والـ GTK والـ Qt وضبط الـ Wayland.

١٦ مايو ٢٠٢٦

لماذا هذه المتغيرات مهمة؟

بدون متغيرات البيئة الصحيحة، تطبيقات مثل Spotify وVSCode والمتصفحات لا تشتغل على Wayland أو تعمل بأداء منخفض عبر XWayland.

ملف env.conf — المتغيرات الأساسية

# في ~/.config/hypr/env.conf أو hyprland.conf

# GTK على Wayland أولاً
env = GDK_BACKEND,wayland,x11

# Qt على Wayland أولاً
env = QT_QPA_PLATFORM,wayland;xcb

# SDL2 (ألعاب وتطبيقات وسائط)
env = SDL_VIDEODRIVER,wayland

# Clutter (بعض تطبيقات GNOME)
env = CLUTTER_BACKEND,wayland

# المسمى الرسمي للـ compositor
env = XDG_CURRENT_DESKTOP,Hyprland
env = XDG_SESSION_TYPE,wayland
env = XDG_SESSION_DESKTOP,Hyprland

# تعطيل window decorations في Qt
env = QT_WAYLAND_DISABLE_WINDOWDECORATION,1

# أداة تخصيص Qt
env = QT_QPA_PLATFORMTHEME,qt5ct

متغيرات Nvidia (إن كنت على كرت Nvidia)

env = LIBVA_DRIVER_NAME,nvidia
env = __GLX_VENDOR_LIBRARY_NAME,nvidia
env = GBM_BACKEND,nvidia-drm
env = __NV_PRIME_RENDER_OFFLOAD,1
⚠️ لا تضف متغيرات Nvidia إن لم يكن لديك كرت Nvidia — ستسبب مشاكل في تطبيقات مثل OBS.

متغيرات تطبيقات Chromium والمتصفحات

# لتشغيل Chromium/Brave/Chrome على Wayland
env = NIXOS_OZONE_WL,1

# Firefox
env = MOZ_ENABLE_WAYLAND,1

متغيرات لا تضبطها بشكل ثابت

هذه المتغيرات تتعارض مع theme switchers الديناميكية:

  • GTK_THEME — اضبط الثيم عبر gsettings أو nwg-look
  • XCURSOR_THEME — استخدم hyprctl setcursor بدلاً منه

أين تضع المتغيرات؟

# في hyprland.conf مباشرة
env = GDK_BACKEND,wayland,x11

# أو في ملف منفصل مصدره hyprland.conf
source = ~/.config/hypr/env.conf
💡 المستخدمون الذين يستخدمون UWSM لتشغيل Hyprland يحتاجون وضع المتغيرات في مكان مختلف — راجع الـ Wiki.
