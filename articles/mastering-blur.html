إتقان تأثير Blur في Hyprland

شرح كامل لجميع إعدادات الـ Blur — من dual-kawase وحجم الـ kernel حتى الـ noise والـ contrast وتأثير الزجاج المصنفر.

١٦ مايو ٢٠٢٦

مقدمة

تأثير الـ Blur الافتراضي في Hyprland مقبول، لكنه مملّ. بضعة إعدادات بسيطة يمكنها تحويله إلى شيء يبدو أكثر ثراءً وحيوية — مثل تأثير الزجاج المصنفر الذي تراه في macOS أو Windows 11. هذه المقالة تشرح كل إعداد متاح في قسم blur حتى تعرف ماذا يفعل كل خيار وكيف تستخدمه.

أين توضع إعدادات الـ Blur؟

إعدادات الـ Blur تقع داخل قسم decoration في ملف إعدادات Hyprland: 

decoration {
    blur {
        enabled = true
        size = 8
        passes = 2
        # ... بقية الإعدادات
    }
}

شرح كل إعداد بالتفصيل

enabled — تفعيل الـ Blur

أول شيء بديهي: تفعيل أو تعطيل تأثير الـ Blur كلياً. Hyprland يستخدم خوارزمية اسمها Dual-Kawase، وهي خوارزمية ضبابية مخصصة طوّرها فريق Hyprland مستوحاةً من مُجمِّع X11 القديم Compton/Picom. تختلف عن Gaussian blur وBox blur من حيث الأداء والنتيجة البصرية. 

enabled = true

size — حجم الـ Kernel

هذا الإعداد يتحكم في حجم kernel الضبابية. ببساطة: تخيّل أن الصورة مكوّنة من pixels، وأن الـ kernel هو شبكة صغيرة (مثلاً 3×3) تُطبَّق على كل pixel. الخوارزمية تأخذ متوسط قيم الـ pixels المحيطة وتضع القيمة الجديدة مكان الـ pixel الأصلي — وهذا ما ينتج الضبابية.

كلما كبر الـ kernel (أي زادت قيمة size)، كلما ازدادت شدة الضبابية:

  • size = 4 — ضبابية خفيفة
  • size = 8 — ضبابية متوازنة (الافتراضي الموصى به)
  • size = 15 — ضبابية كثيفة جداً
size = 8
💡 فكّر في size باعتباره "شدة الضبابية". القيم بين 6 و10 تعطي نتائج متوازنة لمعظم الإعدادات.

passes — عدد التمريرات

Dual-Kawase خوارزمية تكرارية: تطبّق الضبابية مرة، ثم تُعيد تطبيقها على النتيجة، وهكذا. عدد التمريرات يتحكم في كم مرة يحدث هذا:

  • passes = 1 — ضبابية خفيفة مع قليل من التشبّع اللوني
  • passes = 2 — توازن جيد بين الأداء والجودة
  • passes = 4 — تأثير زجاج مصنفر (Frosted Glass) مثل Acrylic في Windows 11
  • passes = 5+ — شفافية عالية جداً، ثقيل على الـ GPU
passes = 2
⚠️ كلما زادت التمريرات، زاد الضغط على الـ GPU. إذا كان جهازك بمواصفات متواضعة، اخفض passes وعوّض بزيادة size.

ignore_opacity — تجاهل شفافية النافذة

بعض النوافذ لها شفافية (opacity) أقل من 1. هذا الإعداد يجعل طبقة الـ Blur تتجاهل تلك الشفافية وتُضبّب كل السطوح بغض النظر، مما يضمن أن تأثير الـ Blur يظهر كما تتوقعه دائماً. 

ignore_opacity = true

new_optimizations — تحسينات الأداء

يُفعّل تحسينات داخلية على خوارزمية الـ Blur للحصول على أداء أفضل مع نفس النتيجة البصرية تقريباً. مُفعَّل افتراضياً ولا داعي لتغييره. 

new_optimizations = true

xray — رؤية الخلفية مباشرةً

بالحالة الافتراضية (xray = false)، النافذة تُضبّب السطح الذي خلفها — أي النوافذ الأخرى أو سطح المكتب. عند تفعيل xray = true، ستُضبّب الخلفية (الـ wallpaper) مباشرةً بدلاً من ذلك.

  • xray = false — يُضبّب ما خلف النافذة (الموصى به عادةً)
  • xray = true — يُضبّب الخلفية دائماً، مهما كان خلف النافذة
xray = false

noise — إضافة حبيبات (Grain)

يُضيف ضجيجاً بصرياً على طبقة الـ Blur، مما يعطي تأثير الزجاج المصنفر الواقعي. القيمة عائمة من 0 إلى 1:

  • noise = 0 — بدون حبيبات (ضبابية ناعمة)
  • noise = 0.05 — حبيبات خفيفة وطبيعية
  • noise = 0.5 — مثل زجاج مصنفر حقيقي
  • noise = 1 — حبيبات كثيفة جداً
noise = 0.05

contrast — التباين

يتحكم في تباين الألوان داخل طبقة الـ Blur — يجعل الداكن أكثر قتامةً والفاتح أكثر إضاءةً. هو السر وراء تأثير الـ Blur المشبع "الحيّ":

  • contrast = 1.0 — لا تغيير في التباين (مسطّح)
  • contrast = 1.5 — تباين لطيف يُعطي عمقاً
  • contrast = 2.0 — تباين حادّ وحيويّ
contrast = 1.5
✨ contrast هو الإعداد الذي يجعل الـ Blur يبدو "مميزاً" بدلاً من مجرد ضبابية عادية. جرّب القيمة 1.5 كنقطة انطلاق.

brightness — السطوع

يتحكم في سطوع منطقة الـ Blur. القيمة الافتراضية 0.8 (أي أغمق قليلاً من الخلفية الأصلية):

  • brightness = 0 — يُطفئ الـ Blur كلياً (لا تفعل هذا)
  • brightness = 0.8 — القيمة الافتراضية، داكن قليلاً
  • brightness = 1.0 — سطوع طبيعي
  • brightness = 2.0 — ساطع جداً، مفيد للثيمات الفاتحة
brightness = 0.8

vibrancy — التشبّع اللوني

يتحكم في مدى حيوية وتشبّع الألوان داخل طبقة الـ Blur. مفيد إذا كنت تريد ألواناً أكثر حيوية أو أقل تشبعاً: 

vibrancy = 0.1696

special — ضبابية الـ Special Workspace

يُفعّل تأثير Blur خلف الـ special workspace عند فتحه. هذا مكلف على الأداء لأنه يُضبّب الشاشة كاملة، لذا يُنصح بإبقائه معطلاً: 

special = false

popups — ضبابية القوائم المنبثقة

يُطبّق الـ Blur على القوائم المنبثقة. في الغالب لا تحتاج لتفعيله لأن القوائم المنبثقة لها حجم صغير والـ Blur قد يجعلها أصعب قراءةً. 

popups = false

الإعدادات الكاملة الموصى بها

هذه مجموعة إعدادات متوازنة تعطي تأثير Blur جميلاً مع أداء مقبول: 

decoration {
    blur {
        enabled = true
        size = 8
        passes = 2
        ignore_opacity = true
        new_optimizations = true
        xray = false
        noise = 0.05
        contrast = 1.5
        brightness = 0.8
        vibrancy = 0.1696
        special = false
        popups = false
    }
}

تأثير الزجاج المصنفر (Frosted Glass)

إذا أردت تأثيراً أقرب لزجاج Windows 11 الـ Acrylic أو macOS، جرّب هذه الإعدادات: 

decoration {
    blur {
        enabled = true
        size = 6
        passes = 4
        ignore_opacity = true
        new_optimizations = true
        xray = false
        noise = 0.08
        contrast = 2.0
        brightness = 0.8
        vibrancy = 0.2
    }
}
💡 passes = 4 مع contrast = 2.0 يعطي تأثير الزجاج المصنفر الأجمل، لكنه أكثر ضغطاً على الـ GPU. اخفض passes إلى 2 إذا لاحظت بطئاً.

ملاحظات مهمة

  • قيمة size وpasses يجب أن تكون 1 على الأقل.
  • كلما زادت قيمة size، كلما احتجت إلى زيادة passes لتجنّب تشويه في حواف الـ Blur.
  • على الأجهزة المتواضعة: اخفض passes وعوّض بزيادة size للحصول على تأثير مقبول بأداء أخف.
  • لا تنسَ تفعيل الشفافية (opacity) على النوافذ التي تريد رؤية الـ Blur فيها — بدون شفافية لن يظهر الـ Blur.
