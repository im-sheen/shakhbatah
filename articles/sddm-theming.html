️ تخصيص SDDM — شاشة تسجيل الدخول

كيف تثبّت وتخصّص شاشة تسجيل الدخول SDDM في Hyprland — من الخلفيات الحية حتى الطقس والثيمات الجاهزة.

١٦ مايو ٢٠٢٦

ما هو SDDM؟

SDDM هو الـ Display Manager الأكثر استخداماً مع Hyprland — هو الشاشة التي تظهر عند تشغيل الجهاز وتطلب منك تسجيل الدخول.

التثبيت

sudo pacman -S sddm
sudo systemctl enable sddm

أين تجد الثيمات

موقع KDE Store يحتوي عشرات الثيمات الجاهزة. البعض يحتاج قدرة على كتابة QML للتعديل.

تثبيت ثيم يدوياً

# استنسخ الثيم
git clone https://github.com/author/sddm-theme.git

# انقله للمكان الصحيح
sudo cp -r sddm-theme /usr/share/sddm/themes/

تفعيل الثيم

افتح /etc/sddm.conf أو أنشئ /etc/sddm.conf.d/theme.conf:

[Theme]
Current=mytheme-name

إضافة عرض الطقس

بعض الثيمات تعرض الطقس الحالي. لإعداد موقعك:

الخطوة 1: ملف إعداد الموقع

# في /etc/sddm-weather.conf
LATITUDE=24.7136
LONGITUDE=46.6753
UNIT=celsius

الخطوة 2: سكريبت تحديث الطقس

أنشئ /usr/local/bin/update-sddm-weather يجلب البيانات من open-meteo.com ويحفظها في /run/sddm-weather.txt.

الخطوة 3: خدمة systemd للتحديث

# /etc/systemd/system/sddm-weather.service
[Unit]
Description=Update SDDM weather

[Service]
Type=oneshot
ExecStart=/usr/local/bin/update-sddm-weather
# /etc/systemd/system/sddm-weather.timer
[Unit]
Description=Update SDDM weather every 10 minutes

[Timer]
OnBootSec=20s
OnUnitActiveSec=10min

[Install]
WantedBy=timers.target
sudo systemctl enable --now sddm-weather.timer

إضافة متغير بيئة لـ SDDM

في /etc/sddm.conf:

[Wayland]
GreeterEnvironment=QML_XHR_ALLOW_FILE_READ=1

هذا المتغير ضروري إن كان الثيم يقرأ ملف الطقس عبر QML XHR.

تخصيص الخطوط

احفظ الخطوط في /usr/local/share/fonts/TTF/ ثم حدّث كاش الخطوط:

sudo fc-cache -r
