إيجاد ثيمات GTK بسهولة

أفضل مصادر للحصول على ثيمات GTK جاهزة — من gnome-look وGitHub حتى ثيمات Kvantum لـ Qt.

١٦ مايو ٢٠٢٦

المصادر الرئيسية

gnome-look.org

gnome-look.org/browse?cat=135 — أكبر مجموعة GTK themes مجانية. ابحث عن الثيم الذي يعجبك، نزّل الملف المضغوط، وفكّ ضغطه في ~/.themes/.

مستودع FCorp على GitHub

github.com/fkorsa — مجموعة ضخمة تشمل Kanagawa وCatppuccin وEverforest وGruvbox وNord وغيرها كثير، كلها GTK3 وGTK4.

Vince Liuice

github.com/vinceliuice — مصمم متخصص بثيمات Gnome المتقنة. أشهر أعماله: Colloid وWhiteSur وGradience.

تثبيت الثيم

# نزّل وفكّ ضغط الثيم
cd ~/Downloads
unzip ThemeName.zip

# ضع الثيم في ~/.themes (ليس /usr/share/themes)
mkdir -p ~/.themes
mv ThemeName ~/.themes/
⚠️ لا تضع الثيمات في /usr/share/themes يدوياً — هذا المجلد يديره pacman وقد يُستبدل عند التحديث. استخدم ~/.themes دائماً.

تطبيق الثيم

# عبر nwg-look (موصى به)
nwg-look

# أو عبر gsettings
gsettings set org.gnome.desktop.interface gtk-theme "ThemeName"

ثيمات Kvantum لتطبيقات Qt

store.kde.org — ثيمات Plasma/Kvantum. بعد التنزيل، افتح Kvantum Manager واختر الثيم.

يمكن أيضاً الحصول على ثيمات Kvantum من Vince Liuice — مثل Graphite الذي يعمل مع أي GTK theme بألوانه المحايدة.

ثيمات أيقونات

# أشهر ثيمات الأيقونات
sudo pacman -S papirus-icon-theme

# ثيم آخر شائع
yay -S tela-icon-theme
