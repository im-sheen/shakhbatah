Title Bars في Hyprland — التثبيت والتبديل

كيف تضيف title bars لنوافذ Hyprland باستخدام إضافة hyprbars مع keybind لتفعيلها وإيقافها حسب الطلب.

١٦ مايو ٢٠٢٦

hyprbars — الإضافة الرسمية

hyprbars هي إضافة رسمية من مستودع Hyprland Plugins تضيف title bars قابلة للتخصيص لنوافذ Hyprland.

التثبيت

# تحديث hyprpm أولاً
hyprpm update

# إضافة المستودع وتثبيت الإضافة
hyprpm add https://github.com/hyprwm/hyprland-plugins
hyprpm enable hyprbars

إعداد الـ hyprbars في الكونفيغ

plugin {
    hyprbars {
        bar_height = 33
        bar_color = rgba(1e1e2eaa)
        bar_blur = true
        bar_part_of_window = true
        bar_precedence_over_border = true

        buttons {
            button_size = 12
            col.maximize = rgba(a6e3a1ff)
            col.close = rgba(f38ba8ff)
        }
    }
}

Toggle — تفعيل وإيقاف بضغطة

بدلاً من تحميل الإضافة دائماً، يمكن تشغيلها وإيقافها بـ keybind — بدون إشعارات مزعجة: 

# في plugins.conf أو binds.conf
$hyprbars = /usr/lib/hyprland/plugins/hyprbars.so

bind = SUPER, Y,       exec, hyprctl plugin load $hyprbars
bind = SUPER SHIFT, Y, exec, hyprctl plugin unload $hyprbars
💡 استخدام hyprctl plugin load/unload أفضل من hyprpm enable/disable لأنه لا يعرض إشعارات ويعمل بسلاسة.

تخصيص أزرار الـ Title Bar

plugin {
    hyprbars {
        # ...
        buttons {
            button_size = 12
            # إغلاق
            col.close = rgba(f38ba8ff)
            # تصغير
            col.minimize = rgba(f9e2afff)
            # تكبير
            col.maximize = rgba(a6e3a1ff)
        }
    }
}
⚠️ إن كنت تستخدم theme switcher، يمكنك تغيير ألوان الأزرار ديناميكياً بتحديث الكونفيغ وإعادة تحميل الإضافة.
