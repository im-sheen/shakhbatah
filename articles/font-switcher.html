بناء قائمة تغيير الخط في Hyprland باستخدام YAD

كيف تبني GUI font switcher باستخدام YAD لعرض نافذة اختيار الخط وتطبيقه على GTK.

١٦ مايو ٢٠٢٦

ما هو YAD؟

YAD (Yet Another Dialog) أداة تتيح بناء نوافذ GUI من Bash — بما فيها نافذة اختيار الخط. مثالية لبناء font switcher بدون كتابة أي كود GTK.

التثبيت

yay -S yad

عرض نافذة اختيار الخط

yad --font

تعرض نافذة اختيار الخط مع قائمة جميع الخطوط المثبّتة وخيار الحجم والـ Style.

سكريبت Font Switcher

#!/bin/bash
# ~/.local/bin/font-switcher.sh

# عرض نافذة اختيار الخط
CHOSEN_FONT=$(yad --font   --title "اختر الخط"   --button="تطبيق:0"   --button="إلغاء:1")

# إن تم اختيار خط
if [ $? -eq 0 ] && [ -n "$CHOSEN_FONT" ]; then
    # تطبيق على GTK3 وGTK4
    gsettings set org.gnome.desktop.interface font-name "$CHOSEN_FONT"
    
    # إشعار بالتغيير
    notify-send "الخط" "تم تطبيق: $CHOSEN_FONT"
fi
chmod +x ~/.local/bin/font-switcher.sh

إضافة window rule لنافذة YAD

# في windowrules.conf
windowrulev2 = float, title:^(اختر الخط)$
windowrulev2 = center, title:^(اختر الخط)$
windowrulev2 = size 600 500, title:^(اختر الخط)$

تطبيق الخط على تطبيقات محددة

بعد اختيار الخط، يمكن تطبيقه على تطبيقات بعينها:

# Waybar — في style.css
* { font-family: "اسم الخط"; }

# Rofi — في ملف الثيم
font: "اسم الخط 11";

# Kitty
font_family اسم الخط

تغيير وزن الخط لـ GTK

تغيير الوزن (Bold/Medium) عبر gsettings لا يعمل دائماً. الطريقة الموثوقة هي gtk.css: 

/* ~/.config/gtk-3.0/gtk.css */
* {
    font-weight: 500;  /* Medium */
}
