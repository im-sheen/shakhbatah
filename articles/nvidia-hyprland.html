كل شيء عن Nvidia في Hyprland

دليل شامل لاستخدام كرت Nvidia مع Hyprland — التثبيت والـ drivers والـ environment variables وأدوات Prime وOptimus.

١٦ مايو ٢٠٢٦

ملاحظة قبل البدء

Hyprland لا يوفر دعماً رسمياً لـ Nvidia، لكن بالإعداد الصحيح تعمل بشكل ممتاز. تجربتك ستكون أفضل بكثير مع الـ drivers الاحتكارية بدلاً من Nouveau مفتوح المصدر.

اختيار الـ Drivers الصحيح

Hyprland يدعم ثلاثة سيناريوهات:

  • Drivers احتكارية كاملة (للكروت القديمة نسبياً)
  • Drivers احتكارية مع open-source kernel modules (مطلوب لكروت 50xx وأحدث)
  • Nouveau مفتوح المصدر (أداء أضعف، غير موصى به)
⚠️ إن كنت على كرت من سلسلة RTX 5000 أو أحدث، يجب استخدام nvidia-open-dkms وليس nvidia-dkms.

التثبيت على Arch Linux

# للكروت الحديثة (50xx وأحدث) — open kernel modules
sudo pacman -S nvidia-open-dkms nvidia-utils nvidia-settings linux-zen linux-zen-headers

# للكروت الأقدم (40xx فما دون) — proprietary drivers
sudo pacman -S nvidia-dkms nvidia-utils nvidia-settings linux-zen linux-zen-headers
💡 نصيحة: kernel لينكس Zen أسرع من الافتراضي — استخدمه إن أمكن.

متغيرات البيئة المطلوبة

أضف هذين السطرين في hyprland.conf أو ملف env.conf:

env = LIBVA_DRIVER_NAME,nvidia
env = __GLX_VENDOR_LIBRARY_NAME,nvidia
⚠️ لا تضف أكثر من اللازم من متغيرات Nvidia — كثرتها تسبب مشاكل مثل عدم اشتغال OBS أو عدم اكتشاف الشاشة.

Prime Run — تشغيل تطبيق محدد بالكرت المخصص

على اللابتوب الذي يحتوي كرت Intel مدمج وNvidia مخصص، يمكنك تشغيل أي تطبيق بقوة الكرت المخصص: 

# تثبيت nvidia-prime
sudo pacman -S nvidia-prime

# تشغيل متصفح بالكرت المخصص
prime-run brave

# تشغيل لعبة بالكرت المخصص
prime-run steam

Super GFXCTL — التحكم في وضع الكرت (لاب توب)

# تثبيت
yay -S supergfxctl

# عرض الوضع الحالي
supergfxctl -g

# عرض الأوضاع المتاحة
supergfxctl -s

# التغيير لوضع مدمج فقط (توفير طاقة)
supergfxctl -m integrated

# العودة لوضع هجين (أداء + توفير)
supergfxctl -m hybrid

إن كان لابتوبك يملك hardware mux switch، ستجد خيار dedicated أيضاً مما يتيح استخدام الكرت المخصص دائماً.

ASUS Laptops — أدوات إضافية

# أدوات ASUS
yay -S asusctl rog-control-center

# تفعيل الخدمة
sudo systemctl enable --now asusd.service

ROG Control Center يتيح ضبط حد الشحن (80% للحفاظ على البطارية) وتغيير platform profile بين quiet وbalanced وperformance.

تحديد Flickering في تطبيقات Electron

إن رأيت وميضاً في تطبيقات مثل VSCode أو Spotify، راجع قسم البيئة في Hyprland Wiki واضبط الـ environment variables المناسبة لـ Electron على Nvidia.
