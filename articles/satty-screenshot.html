Satty — أداة التعليق على لقطات الشاشة

كيف تثبّت وتستخدم Satty مع Hyprshot لالتقاط لقطات شاشة وإضافة تعليقات وأسهم.

١٦ مايو ٢٠٢٦

ما هو Satty؟

Satty أداة تعليق على لقطات الشاشة مبنية بـ Rust لـ Wayland — تفتح تلقائياً بعد أخذ لقطة الشاشة وتتيح إضافة أسهم ونصوص ومستطيلات وتمييز وطمس.

التثبيت

sudo pacman -S satty hyprshot

ربط مع Hyprshot

# في binds.conf

# لقطة شاشة كاملة
bind = , Print, exec, hyprshot -m output --raw | satty --filename -

# لقطة منطقة مخصصة
bind = SHIFT, Print, exec, hyprshot -m region --raw | satty --filename -

# لقطة النافذة الحالية
bind = SUPER, Print, exec, hyprshot -m window --raw | satty --filename -

عند الانتهاء

  • Copy — نسخ للحافظة مباشرة (الأكثر استخداماً)
  • Save — حفظ في ملف
💡 --filename - في الأمر يعني "اقرأ الصورة من stdin" — هكذا تُمرَّر لقطة hyprshot مباشرة لـ Satty دون حفظ ملف مؤقت.

تخصيص Satty

# ~/.config/satty/config.toml
[general]
default_tool = "Brush"
early_exit = true          # أغلق بعد النسخ/الحفظ
copy-command = "wl-copy"
