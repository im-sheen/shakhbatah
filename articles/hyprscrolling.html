↔️ Hyprscrolling — تخطيط Niri داخل Hyprland

كيف تثبّت وتضبط إضافة hyprscrolling للحصول على تخطيط التمرير الأفقي اللانهائي من Niri.

١٦ مايو ٢٠٢٦

ما هو Hyprscrolling؟

Hyprscrolling إضافة لـ Hyprland تضيف تخطيط Scrolling — بدلاً من workspaces منفصلة، لديك مستوى أفقي لانهائي تتمرر عليه النوافذ جنباً إلى جنب. هذا هو نفس تخطيط Niri الشهير.

المتطلب: hyprland-git

الإضافات تتطلب hyprland-git وليس نسخة stable. مع Chaotic AUR يمكن تثبيته مباشرة: 

sudo pacman -S hyprland-git aquamarine-git hyprlang-git

تثبيت HyprPM ثم الإضافة

# تحديث HyprPM
hyprpm update

# إضافة مستودع الإضافات
hyprpm add https://github.com/hyprwm/hyprland-plugins

# تثبيت وتفعيل hyprscrolling
hyprpm enable hyprscrolling

تفعيل التخطيط

في hyprland.conf:

general {
    layout = scrolling
}

إعداد خيارات hyprscrolling

plugin {
    hyprscrolling {
        # تكبير نافذة وحيدة لتملأ الشاشة
        fullscreen_on_one_column = true

        # المركزة على النافذة المفعّلة
        focus_fit_method = center

        # متابعة التركيز (false = انقر للتركيز)
        follow_focus = false

        # عرض الأعمدة الافتراضي
        column_widths = oneeighth onesixth onefourth onethird onehalf twothirds
    }
}

التنقل بين النوافذ

# في binds.conf
bind = SUPER, left,  movefocus, l
bind = SUPER, right, movefocus, r
bind = SUPER, up,    movefocus, u
bind = SUPER, down,  movefocus, d

# تغيير عرض العمود
bind = SUPER, minus, scroller:cyclesize, -1
bind = SUPER, equal, scroller:cyclesize, 1
⚠️ hyprscrolling لا يزال قيد التطوير وقد تظهر بعض الـ bugs. لتعطيل bug التكبير التلقائي، فعّل fullscreen_on_one_column = true.
