تثبيت تطبيقات AUR بدون ترجمة — Chaotic AUR

كيف تضيف Chaotic AUR لـ Arch Linux لتثبيت أشهر تطبيقات AUR كبرامج جاهزة بدون Compile.

١٦ مايو ٢٠٢٦

المشكلة

تطبيقات AUR تتطلب عادةً تنزيل الكود المصدري وترجمته على جهازك — وهذا يستغرق وقتاً طويلاً ويحمّل المعالج بشكل كبير. Brave Browser مثلاً يحتاج ساعة للترجمة.

الحل: Chaotic AUR

Chaotic AUR مستودع يُبني فيه أشهر تطبيقات AUR بشكل تلقائي ويوفّر الـ binary — مما يجعل التثبيت فورياً تماماً مثل تطبيقات extra.

الإضافة (خطوتان)

# الخطوة 1: إضافة المفتاح والـ keyring
sudo pacman-key --recv-key 3056513887B78AEB --keyserver keyserver.ubuntu.com
sudo pacman-key --lsign-key 3056513887B78AEB
sudo pacman -U 'https://cdn-mirror.chaotic.cx/chaotic-aur/chaotic-keyring.pkg.tar.zst'
sudo pacman -U 'https://cdn-mirror.chaotic.cx/chaotic-aur/chaotic-mirrorlist.pkg.tar.zst'
# الخطوة 2: إضافة المستودع لـ /etc/pacman.conf
sudo tee -a /etc/pacman.conf << 'EOF'

[chaotic-aur]
Include = /etc/pacman.d/chaotic-mirrorlist
EOF
# تحديث قائمة الحزم
sudo pacman -Sy

الاستخدام

بعد الإضافة، يمكنك تثبيت أي حزمة Chaotic AUR مباشرة بـ pacman أو yay:

# مثال: Hyprland Git
sudo pacman -S hyprland-git

# مثال: auto-cpufreq
sudo pacman -S auto-cpufreq

# مثال: Brave bin (بدون ترجمة)
sudo pacman -S brave-bin
💡 الحزم الموجودة في Chaotic AUR تظهر باللون الأخضر مثل حزم extra — بدلاً من الأخضر الفاتح الخاص بـ AUR.

ماذا يقدم Chaotic AUR؟

يحتوي على أكثر من 3000 حزمة شائعة مثل: hyprland-git، auto-cpufreq، brave-bin، spotify، vesktop (Discord)، وكثير غيرها.
