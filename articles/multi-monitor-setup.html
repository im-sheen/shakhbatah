️ إعداد شاشات متعددة في Hyprland

الطريقة الأسهل لإعداد Multi-Monitor في Hyprland — من معرفة اسم الشاشة حتى ربط الـ workspaces وإخفاء الشاشات وعكسها.

١٦ مايو ٢٠٢٦

مقدمة

إعداد شاشات متعددة في Hyprland أسهل مما تتوقع. كل شيء يتم عبر ملف config واحد، وبسطرين فقط تكون شاشتك الثانية تعمل بالضبط كما تريد. في هذا الدليل سنمر على كل شيء: من إيجاد اسم الشاشة، إلى الدقة والموضع والـ scale، وصولاً لربط الـ workspaces وعكس الشاشات.

أين تكتب إعداد الشاشات؟

يمكنك كتابة إعداد الشاشات مباشرة داخل hyprland.conf، لكن الأفضل فصله في ملف مستقل لتنظيم أفضل. مثلاً: 

~/.config/hypr/modules/monitors.conf

ثم تضيف في hyprland.conf: 

source = ~/.config/hypr/modules/monitors.conf

هذه طريقة ممتازة لتقسيم الكونفيج الكبير إلى ملفات صغيرة يسهل التعامل معها.

الصيغة الأساسية

كل شاشة تُعرَّف بسطر واحد بهذا الشكل: 

monitor = اسم_الشاشة, الدقة@معدل_التحديث, الموضع, الـscale

مثال حقيقي:

monitor = DP-1,    2560x1440@143.97, 0x0,    1
monitor = eDP-1,   1920x1200@60,     2560x0,  1.25
💡 الترتيب مهم: الاسم ← الدقة@معدل التحديث ← الموضع X و Y ← الـ Scale

الخطوة الأولى: اعرف اسم شاشتك

قبل أي شيء، تحتاج معرفة الاسم الذي يعرفه Hyprland لشاشتك. الأمر بسيط: 

hyprctl monitors

هذا سيعطيك قائمة بكل الشاشات المتصلة. ستجد أسماءً مثل:

  • eDP-1 — الشاشة المدمجة في اللابتوب
  • DP-1 أو DP-2 — شاشة خارجية عبر DisplayPort
  • HDMI-A-1 — شاشة خارجية عبر HDMI

ستجد أيضاً جميع الدقات ومعدلات التحديث المتاحة لكل شاشة تحت available modes.

تحديد الدقة ومعدل التحديث

انسخ الدقة ومعدل التحديث بالضبط كما تظهر في نتيجة hyprctl monitors. إذا كانت الشاشة تدعم 143.97 Hz، اكتبها 143.97 وليس 144 — الفرق الصغير يسبب مشاكل. 

monitor = DP-1, 2560x1440@143.97, 0x0, 1

تحديد الموضع (Position)

الموضع هو إحداثيات X و Y لبداية الشاشة. الشاشة الأولى تبدأ عند 0x0. الشاشة الثانية تبدأ عند نهاية الأولى.

مثال: عندي شاشة رئيسية عرضها 2560 بكسل. إذاً الشاشة الثانية تبدأ عند 2560x0: 

monitor = DP-1,  2560x1440@143.97, 0x0,    1      # الشاشة الرئيسية (يسار)
monitor = eDP-1, 1920x1200@60,     2560x0,  1.25   # اللابتوب (يمين)
💡 إحداثيات Y لا تتغير إذا أردت الشاشات في سطر أفقي واحد. غيّرها فقط إذا أردت شاشة فوق الأخرى.

الـ Scale (حجم العناصر)

الـ Scale يتحكم في حجم العناصر على الشاشة. القيمة 1 تعني الحجم الطبيعي. 1.25 يكبّر العناصر 25%. 

monitor = eDP-1, 1920x1200@60, 2560x0, 1.25
⚠️ تنبيه: تغيير الـ Scale قد يجعل بعض تطبيقات Xwayland تبدو ضبابية أو مكسورة، لأنها لا تدعم Wayland بشكل أصيل. تطبيقات Wayland الأصيلة لا تتأثر.

إيقاف تشغيل شاشة

تريد إيقاف شاشة مؤقتاً دون فصلها؟ ببساطة: 

monitor = eDP-1, disable

فور حفظ الكونفيج، الشاشة تطفئ على الفور. لإعادة تشغيلها، احذف هذا السطر أو ضع الإعداد الطبيعي مرة أخرى.

عكس الشاشة (Mirroring)

تريد أن تظهر نفس صورة شاشة على شاشة أخرى؟ استخدم mirror: 

monitor = DP-2, 1920x1080@60, 0x0, 1, mirror, DP-1

هذا يعني: اعرض على DP-2 نفس ما يظهر على DP-1.

⚠️ المحاذير: العكس لا يعيد رسم الصورة. إذا عكست شاشة 1080p على شاشة 4K، ستظل الدقة 1080p. وإذا كانت نسب العرض مختلفة ستظهر أشرطة سوداء على الجوانب.

خيارات متقدمة

شاشة 10-bit (ألوان أوسع)

إذا كانت شاشتك تدعم 10-bit (مليار لون بدلاً من 16 مليون): 

monitor = DP-1, 2560x1440@143.97, 0x0, 1, bitdepth, 10

مفيد للمحررين والمصممين. لكن بعض التطبيقات قد لا تتوافق معه، اختبر قبل الاعتماد عليه.

صيغة monitor v2 (أوضح للقراءة)

بدلاً من سطر واحد طويل، يمكنك كتابة إعداد الشاشة بصيغة block: 

monitor {
    output   = DP-1
    mode     = 2560x1440@143.97
    position = 0x0
    scale    = 1
}

النتيجة نفسها تماماً، لكن أسهل في القراءة والتعديل.

ربط الـ Workspaces بشاشة معينة

هذه من أقوى ميزات الشاشات المتعددة في Hyprland. يمكنك تخصيص workspaces معينة لشاشة معينة، بحيث عند الانتقال إليها تفتح على الشاشة الصحيحة تلقائياً. 

# workspaces 1-5 على الشاشة الرئيسية
workspace = 1, monitor:DP-1
workspace = 2, monitor:DP-1
workspace = 3, monitor:DP-1
workspace = 4, monitor:DP-1
workspace = 5, monitor:DP-1

# workspaces 6-10 على اللابتوب
workspace = 6,  monitor:eDP-1
workspace = 7,  monitor:eDP-1
workspace = 8,  monitor:eDP-1
workspace = 9,  monitor:eDP-1
workspace = 10, monitor:eDP-1

بعد هذا الإعداد، عند الانتقال للـ workspace 6 وفتح نافذة، ستفتح مباشرة على شاشة اللابتوب. حتى شريط Waybar ومؤشر الـ workspace النشط سيتحرك للشاشة الصحيحة.

ملف monitors.conf كامل

هذا مثال كامل جاهز للاستخدام يجمع كل ما ذكرناه:

# ─── monitors.conf ───────────────────────────────
# الشاشة الرئيسية: 2.5K خارجية
monitor = DP-1, 2560x1440@143.97, 0x0, 1

# شاشة اللابتوب: على اليمين
monitor = eDP-1, 1920x1200@60, 2560x0, 1.25

# ─── Workspace Bindings ──────────────────────────
workspace = 1,  monitor:DP-1
workspace = 2,  monitor:DP-1
workspace = 3,  monitor:DP-1
workspace = 4,  monitor:DP-1
workspace = 5,  monitor:DP-1

workspace = 6,  monitor:eDP-1
workspace = 7,  monitor:eDP-1
workspace = 8,  monitor:eDP-1
workspace = 9,  monitor:eDP-1
workspace = 10, monitor:eDP-1

خلاصة — الأوامر المهمة

  • hyprctl monitors — اعرف أسماء الشاشات والدقات المتاحة
  • monitor = اسم, دقة@hz, موضع, scale — الصيغة الأساسية
  • monitor = اسم, disable — أوقف تشغيل الشاشة
  • monitor = اسم, ..., mirror, اسم_آخر — اعكس شاشة على أخرى
  • workspace = رقم, monitor:اسم — اربط workspace بشاشة
