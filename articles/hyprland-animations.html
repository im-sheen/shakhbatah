️ كيف تخصّص الأنيميشن في Hyprland

شرح كامل لنظام الأنيميشن — من البيزيير كيرف حتى أنيميشن النوافذ والـ workspaces والـ layers.

١٦ مايو ٢٠٢٦

بنية الأنيميشن

كل سطر أنيميشن في Hyprland يتبع نفس الصيغة: 

animation = name, on/off, speed, curve, style
  • name — اسم الأنيميشن (windows, workspaces, layers…)
  • on/off — 1 لتفعيل، 0 لتعطيل
  • speed — السرعة، القيمة الافتراضية حوالي 7
  • curve — اسم البيزيير كيرف التي تريد استخدامها
  • style — نوع الحركة (slide, popin, fade…)

البيزيير كيرف — ما هي؟

البيزيير كيرف هي مجموعة من التعليمات الرياضية تحدد شكل حركة الأنيميشن. تُعرَّف في الكونفيغ هكذا: 

bezier = myBezier, 0.05, 0.9, 0.1, 1.05

الأرقام الأربعة هي نقاط التحكم في المنحنى — النقطتان الأولى والأخيرة تحددان نقطة البداية والنهاية، والنقطتان الوسطى تعملان كـ"مغناطيسات" تشكّل منحنى الحركة.

💡 لتصميم بيزيير كيرف مخصصة جرّب موقع cssportal.com أو easings.net — تقدر تجرب المنحنيات بصرياً وتنسخ الإحداثيات مباشرة.

أنيميشن النوافذ (Windows)

للنوافذ ثلاثة أنواع من الـ style:

  • slide — تنزلق النافذة من الجانب
  • popin — تظهر النافذة من نقطة صغيرة وتكبر. يمكن تحديد النسبة المئوية للبداية: popin 80%
  • gnomed — مستوحى من GNOME، تتمدد النافذة من الأعلى والأسفل
animation = windowsIn,  1, 7, myBezier, slide
animation = windowsOut, 1, 7, myBezier, slide
animation = windowsMove,1, 6, myBezier
💡 windowsMove يتحكم في حركة النافذة أثناء السحب وتغيير الحجم. يمكن استخدام بيزيير مختلفة لجعل هذه الحركة أكثر سلاسة.

أنيميشن الـ Layers

الـ layers هي العناصر التي تعلو النوافذ كـ Waybar وRofi — Hyprland لا يعاملها كنوافذ عادية. لها نفس الـ styles: 

animation = layersIn,  1, 7, myBezier, slide
animation = layersOut, 1, 7, myBezier, slide
⚠️ إذا كان لديك layerrule يحدد أنيميشن خاص بـ Rofi مثلاً، فسيتجاوز الإعداد العام. اضبطه أو احذفه لرؤية التأثير الصحيح.

أنيميشن الـ Workspaces

للـ workspaces styles أكثر تنوعاً:

Styleالوصف
slideانزلاق أفقي (الافتراضي)
slidevertانزلاق عمودي — مناسب لمن يستخدم شريطاً جانبياً
fadeتلاشي بين الـ workspaces
slidefadeانزلاق مع تلاشٍ في نفس الوقت
slidefadevertنفس السابق لكن عمودياً
 
animation = workspaces, 1, 8, myBezier, slide

لـ slidefade يمكن تحديد النسبة المئوية للحركة قبل التلاشي: 

animation = workspaces, 1, 8, myBezier, slidefade 20%

أنيميشن الـ Fade والـ Border

# تلاشي النوافذ عند الفتح والإغلاق
animation = fadeIn,  1, 5, myBezier
animation = fadeOut, 1, 5, myBezier

# تدوير لون الـ border تلقائياً (RGB effect)
animation = border,      1, 10, myBezier
animation = borderangle, 1, 8,  myBezier

مثال كامل جاهز للاستخدام

animations {
    enabled = true

    bezier = myBezier, 0.05, 0.9, 0.1, 1.05
    bezier = linear,   0.0,  0.0, 1.0, 1.0

    animation = windows,     1, 7,  myBezier, slide
    animation = windowsIn,   1, 7,  myBezier, slide
    animation = windowsOut,  1, 7,  myBezier, slide
    animation = windowsMove, 1, 6,  linear

    animation = layersIn,    1, 7,  myBezier, slide
    animation = layersOut,   1, 7,  myBezier, slide

    animation = workspaces,  1, 8,  myBezier, slidefade 15%
    animation = specialWorkspace, 1, 8, myBezier, slidevert

    animation = fadeIn,      1, 5,  myBezier
    animation = fadeOut,     1, 5,  myBezier
    animation = border,      1, 10, myBezier
    animation = borderangle, 1, 8,  linear
}

تخصيص اتجاه الـ Slide

يمكنك إجبار الـ slide على اتجاه معين: 

animation = layersIn,  1, 7, myBezier, slide left
animation = layersOut, 1, 7, myBezier, slide right
