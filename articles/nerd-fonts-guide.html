استخدام Nerd Fonts بالطريقة الصحيحة في Hyprland

الفرق بين نسخ Regular وMono وPropo — وأيها تستخدم في Waybar مقابل التيرمنال.

١٦ مايو ٢٠٢٦

المشكلة الشائعة

إن لاحظت أيقونات Nerd Font غير محاذية بشكل صحيح في Waybar — إما متلاصقة مع النص أو بينها مسافة كبيرة — فالسبب هو اختيار نسخة الخط الخاطئة.

أنواع نسخ Nerd Fonts

النسخةالاستخدامالوصف
Regularعامالنسخة الأصلية بدون تعديلات على التباعد
Monoالتيرمنالأيقونات أصغر، مثالية لنصوص monospace
PropoWaybar والـ UIتباعد proportional — الأيقونات تظهر بحجمها الصحيح في السياقات غير-monospace

القاعدة العملية

  • Waybar وRofi وأي واجهة UI → استخدم نسخة Propo
  • التيرمنال (Kitty أو Foot) → استخدم النسخة العادية أو Mono

مثال: Waybar

/* في style.css */
* {
    font-family: "JetBrainsMono Nerd Font Propo", "SF Pro Display";
}

مثال: Kitty

# في kitty.conf
font_family      JetBrainsMono Nerd Font
# أو
font_family      GeistMono Nerd Font

تحميل Nerd Fonts

# من AUR (مثال: JetBrains Mono)
yay -S ttf-jetbrains-mono-nerd

# أو من الموقع الرسمي
# nerdfonts.com → Downloads → اختر الخط

معاينة الأيقونات

موقع nerdfonts.com/cheat-sheet يعرض جميع الأيقونات مع أسمائها وكود Unicode الخاص بها.

💡 بمجرد اختيار نسخة Propo لـ Waybar، ستختفي مشكلة التباعد تلقائياً دون الحاجة لإضافة مسافات يدوياً في الكونفيغ.
