Hyprsunset — وضع الليل في Hyprland

كيف تضبط Hyprsunset لتقليل درجة حرارة اللون ليلاً مع keybind للتبديل وجدول تلقائي.

١٦ مايو ٢٠٢٦

ما هو Hyprsunset؟

Hyprsunset هو أداة من نظام Hyprland البيئي لضبط درجة حرارة اللون (Color Temperature) — يجعل الشاشة أدفأ (أكثر صفرة) ليلاً لتقليل الإجهاد البصري.

التثبيت

sudo pacman -S hyprsunset

الاستخدام المباشر

# تشغيل وضع الليل (3000 كلفن = دافئ جداً)
hyprsunset -t 3000

# إيقاف تشغيله
pkill hyprsunset

Toggle Script — تشغيل وإيقاف بضغطة

أنشئ ~/.local/bin/night-mode.sh:

#!/bin/bash
if pgrep -x hyprsunset > /dev/null; then
    pkill hyprsunset
    notify-send "🌙 وضع الليل" "تم الإيقاف"
else
    hyprsunset -t 3200 &
    notify-send "🌙 وضع الليل" "تم التفعيل"
fi
chmod +x ~/.local/bin/night-mode.sh

أضف keybind في binds.conf:

bind = SUPER, N, exec, night-mode.sh

جدول تلقائي حسب الوقت

أضف في ~/.config/hypr/hyprsunset.conf:

# نهاراً — لا تغيير
profile {
    time = 07:30
    identity = true
}

# ليلاً — درجة حرارة دافئة
profile {
    time = 21:00
    temperature = 3200
    gamma = 90
}

ثم شغّله في autostart:

exec-once = hyprsunset

الـ Gamma

--gamma يتحكم في السطوع الظاهري (Perceived Brightness) بدون تغيير السطوع الفعلي. القيمة الافتراضية 100% — الخفض يجعل الشاشة تبدو أكثر قتامة. 

hyprsunset -t 4000 --gamma 85
💡 يمكن التحكم في Hyprsunset أيضاً عبر IPC باستخدام hyprctl لأتمتة إضافية.
