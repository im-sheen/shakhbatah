اجعل Hyprland أسرع بثلاث أدوات

ثلاث طرق لتسريع Hyprland بشكل ملحوظ — auto-cpufreq للمعالج وSuper GFXCTL للـ GPU وإيقاف الأنيميشن.

١٦ مايو ٢٠٢٦

الطريقة الأولى: auto-cpufreq

auto-cpufreq يضبط تردد المعالج تلقائياً حسب الحِمل — يرفعه عند الحاجة للأداء ويخفضه لتوفير الطاقة. 

# من Chaotic AUR (بدون ترجمة)
sudo pacman -S auto-cpufreq

# تفعيل الخدمة
sudo systemctl enable --now auto-cpufreq

لمتابعة الحالة مباشرة:

auto-cpufreq --stats
⚠️ لا تستخدم auto-cpufreq وpowerprofiled في نفس الوقت — سيتعارضان على التحكم بالمعالج. اختر واحداً فقط.

الطريقة الثانية: power-profiles-daemon

بديل أبسط لـ auto-cpufreq، مشابه لما يستخدمه GNOME وKDE:

sudo pacman -S power-profiles-daemon
sudo systemctl enable --now power-profiles-daemon

# عرض الوضع الحالي
powerprofilesctl list

# تفعيل وضع الأداء
powerprofilesctl set performance

# توفير طاقة
powerprofilesctl set power-saver

الطريقة الثالثة: Super GFXCTL (لأجهزة Nvidia)

yay -S supergfxctl

# عرض الأوضاع المتاحة
supergfxctl -s

# التبديل لوضع dedicated (إن توفر hardware mux switch)
supergfxctl -m dedicated

بونس: إيقاف الأنيميشن والتأثيرات في وضع الألعاب

أسرع طريقة لتحسين الأداء في Hyprland هي إيقاف الأنيميشن وBlur والظلال مؤقتاً:

# في binds.conf — toggle لوضع الألعاب
bind = SUPER, F1, exec, hyprctl keyword animations:enabled false && hyprctl keyword decoration:blur:enabled false && hyprctl keyword decoration:shadow:enabled false

bind = SUPER, F2, exec, hyprctl keyword animations:enabled true && hyprctl keyword decoration:blur:enabled true && hyprctl keyword decoration:shadow:enabled true
💡 يمكنك بناء "game mode" كامل: يوقف الأنيميشن، يُبدّل layout الـ Waybar، ويُوقف Blur كله دفعة واحدة.
