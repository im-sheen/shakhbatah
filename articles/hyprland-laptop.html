إعداد Hyprland للاب توب — الدليل الشامل

كيف تضبط Hyprland للاب توب بشكل صحيح — إدارة الطاقة والـ GPU والـ VRR وإعدادات الأداء على البطارية.

١٦ مايو ٢٠٢٦

أولاً: إدارة الطاقة

اختر أحد الأداتين — لا تستخدمهما معاً:

الخيار الأول: auto-cpufreq (موصى به)

sudo pacman -S auto-cpufreq
sudo systemctl enable --now auto-cpufreq

يضبط التردد تلقائياً — أداء مرتفع عند الشحن، توفير عند البطارية.

الخيار الثاني: power-profiles-daemon

sudo pacman -S power-profiles-daemon
sudo systemctl enable --now power-profiles-daemon

# ضبط يدوي
powerprofilesctl set balanced    # عادي
powerprofilesctl set performance # أداء
powerprofilesctl set power-saver # توفير

ثانياً: إعداد Nvidia (إن وُجد)

yay -S supergfxctl

# عرض الوضع الحالي
supergfxctl -g

# وضع هجين (موصى به على البطارية)
supergfxctl -m hybrid

# وضع مدمج فقط (أقصى توفير للطاقة)
supergfxctl -m integrated

استخدم prime-run app لتشغيل أي تطبيق بالكرت المخصص عند الحاجة.

ثالثاً: لأجهزة ASUS

yay -S asusctl rog-control-center
sudo systemctl enable --now asusd.service
  • اضبط حد الشحن على 80% للحفاظ على عمر البطارية
  • غيّر platform profile حسب الحاجة (quiet / balanced / performance)

رابعاً: إعدادات Hyprland الخاصة باللاب توب

# في hyprland.conf أو misc.conf

# Variable Refresh Rate — يوفر طاقة
misc {
    vrr = 1
}

# إيقاف الأنيميشن لأقصى أداء
animations {
    enabled = false
}

# استخدام تيرمنال خفيف
# foot بدلاً من kitty يشتغل أسرع

خامساً: إعداد Hypridle للإضاءة والقفل

listener {
    timeout = 150
    on-timeout = brightnessctl set 10
    on-resume = brightnessctl set 100%
}

listener {
    timeout = 300
    on-timeout = hyprlock
}

listener {
    timeout = 380
    on-timeout = systemctl suspend
}
