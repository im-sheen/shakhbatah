إنشاء Modules مخصصة في Waybar

كيف تبني custom modules في Waybar من الصفر — مع أمثلة عملية لعداد التحديثات ومعلومات الميديا.

١٦ مايو ٢٠٢٦

المفهوم الأساسي

الـ custom module في Waybar هو وحدة تنفذ أمراً shell أو سكريبت وتعرض مخرجاته في الشريط. تُعرَّف في ~/.config/waybar/config.jsonc هكذا: 

"custom/mymodule": {
    "exec": "some-command",
    "interval": 60,
    "format": "📦 {}",
    "tooltip": true
}

الخصائص الأساسية

الخاصيةالوصف
execالأمر أو السكريبت الذي يُنفَّذ
exec-ifينفذ الـ exec فقط إن أعاد هذا الأمر 0
intervalالتحديث كل N ثانية — استخدم once لمرة واحدة
formatالشكل — {} يُستبدل بمخرج الأمر
on-clickأمر ينفذ عند النقر بالزر الأيسر
return-typeاضبطه على json إن أعاد السكريبت JSON
tooltip-formatنص الـ tooltip عند التحويم

مثال 1: عداد تحديثات Pacman

"custom/updates": {
    "exec": "checkupdates | wc -l",
    "exec-if": "checkupdates",
    "interval": "once",
    "format": "🔄 {} تحديث",
    "on-click": "kitty --hold yay -Syu",
    "tooltip-format": "{}",
    "signal": 8
}
💡 تحتاج pacman-contrib لأمر checkupdates:
sudo pacman -S pacman-contrib

مثال 2: معلومات الميديا الجارية

"custom/media": {
    "format": "{}",
    "escape": true,
    "return-type": "json",
    "max-length": 40,
    "on-click": "playerctl play-pause",
    "on-click-right": "playerctl stop",
    "smooth-scrolling-threshold": 1,
    "on-scroll-up": "playerctl next",
    "on-scroll-down": "playerctl previous",
    "exec": "~/.config/waybar/scripts/media-player.py 2>/dev/null"
}

يعتمد هذا على سكريبت Python موجود في مستودع Waybar تحت resources/custom-modules/media-player.py — احفظه في ~/.config/waybar/scripts/.

مثال 3: module مخصص من الصفر

مثال يعرض عدد الملفات في مجلد Home:

"custom/homecount": {
    "exec": "ls -1 ~ | wc -l",
    "interval": "once",
    "format": "🏠 {}",
    "tooltip-format": "{} ملف في Home"
}

إضافة الـ Module للشريط

أضف اسم الـ module في قائمة modules-right أو modules-center أو modules-left:

"modules-right": ["custom/updates", "custom/media", "clock"]

تنسيق CSS للـ Module

#custom-updates {
    color: #89b4fa;
    background: rgba(137, 180, 250, 0.1);
    border-radius: 8px;
    padding: 0 10px;
}
