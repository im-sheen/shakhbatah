️ Alt+Tab احترافي في Hyprland مع Snappy Switcher

كيف تثبّت وتضبط Snappy Switcher للحصول على alt+tab سريع وجميل في Hyprland.

١٦ مايو ٢٠٢٦

ما هو Snappy Switcher؟

Snappy Switcher أداة Alt+Tab حديثة لـ Wayland تعمل على معمارية client-server — مما يجعلها سريعة جداً. تدعم ثيمات INI قابلة للتخصيص وعرض النوافذ بطريقة Overview أو Context.

التثبيت

yay -S snappy-switcher

الإعداد الأولي

snappy install-config

إضافة autostart

# في autostart.conf
exec-once = snappy-switcher --daemon

ربط الـ keybinds

# في binds.conf
bind = ALT, Tab,       exec, snappy-switcher next
bind = ALT SHIFT, Tab, exec, snappy-switcher prev

خيارات الكونفيغ الرئيسية

ملف الكونفيغ في ~/.config/snappy-switcher/config.toml:

[general]
# كيف تُعرض النوافذ: overview أو context
view_mode = "context"

# متابعة الشاشة الأحدث
follow_monitor = true

[theme]
# اسم الثيم
name = "catppuccin-mocha"

[layout]
card_width = 200
card_height = 150
spacing = 10
padding = 20
max_cols = 6
icon_size = 64
corner_radius = 12

[font]
family = "SF Pro Text"
style = "Medium"
title_size = 11.5

الثيمات المتاحة

Gruvbox Dark، Rosé Pine، Catppuccin Mocha، Tokyo Night، Nord، Everforest — وغيرها. تتوفر كملفات INI يمكن تعديلها بسهولة.

وضع Overview مقابل Context

  • Context — يجمع نوافذ نفس التطبيق في بطاقة واحدة (مثل macOS)
  • Overview — نافذة مستقلة = بطاقة مستقلة (مثل Windows 11)
