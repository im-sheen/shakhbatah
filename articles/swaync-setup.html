إعداد وتخصيص SwayNC

كيف تضبط مركز الإشعارات SwayNC — من تثبيت كونفيغ جاهز وتعديله حتى ربطه بألوان Matugen.

١٦ مايو ٢٠٢٦

ما هو SwayNC؟

SwayNC هو daemon للإشعارات لـ Wayland — يعرض الإشعارات ويوفّر مركز تحكم يمكن فتحه بـ keybind لرؤية كل الإشعارات السابقة مع أزرار سريعة للتحكم بالإعدادات.

التثبيت

sudo pacman -S swaync

أضفه في autostart:

exec-once = swaync

إيجاد كونفيغ جاهز

أفضل طريقة للبداية هي العثور على كونفيغ يعجبك من discussions صفحة SwayNC على GitHub، ثم تعديله ليتناسب مع إعدادك. ابحث عن "Show off your swaync" في صفحة الـ discussions. 

# استنسخ أي كونفيغ يعجبك في /tmp
cd /tmp
git clone https://github.com/author/dotfiles.git

# انسخ مجلد swaync
cp -r dotfiles/.config/swaync ~/.config/

هيكل ملفات الكونفيغ

~/.config/swaync/
├── config.json       # الكونفيغ الرئيسي
├── style.css         # التنسيق
└── themes/           # ثيمات (اختياري)

ربط الألوان مع Matugen

في ~/.config/matugen/config.toml:

[[config.templates]]
input_path = "~/.config/matugen/templates/swaync-colors.css"
output_path = "~/.config/swaync/matugen-colors.css"

ثم في style.css:

@import "matugen-colors.css";

إضافة keybind لفتح المركز

bind = SUPER, N, exec, swaync-client -t

تغيير موضع المركز

في config.json:

{
  "positionX": "left",
  "positionY": "top"
}
💡 مجلد /tmp يُمسح عند كل إعادة تشغيل — مثالي لاستنساخ ملفات مؤقتة لا تريد الاحتفاظ بها.
