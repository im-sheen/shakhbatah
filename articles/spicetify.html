تخصيص Spotify مع Spicetify

كيف تثبّت Spicetify وتغيّر مظهر Spotify لأي ثيم تريده — من التثبيت حتى تطبيق الثيمات والألوان.

١٦ مايو ٢٠٢٦

ما هو Spicetify؟

Spicetify هو أداة command-line تتيح لك تخصيص واجهة Spotify بالكامل — تغيير الثيمات والألوان وحتى إضافة ميزات جديدة.

التثبيت

# على Arch Linux
yay -S spicetify-cli

الإعداد الأولي

# توليد ملف الكونفيغ
spicetify

# تفعيل أدوات المطور (مطلوب لبعض الثيمات)
spicetify enable-devtools

تثبيت مجموعة الثيمات الرسمية

# استنساخ مجموعة الثيمات
git clone https://github.com/spicetify/spicetify-themes.git

# نسخها للمجلد الصحيح
cp -r spicetify-themes/* ~/.config/spicetify/Themes/

أو إن كنت على Arch:

yay -S spicetify-themes-git

تطبيق ثيم

# اختيار الثيم
spicetify config current_theme Sleek

# اختيار نظام الألوان
spicetify config color_scheme BladeRunner

# تطبيق التغييرات
spicetify apply

أبرز الثيمات المتاحة

الثيمالوصف
Sleekتصميم أنيق ومتعدد الألوان
Comfyمريح وهادئ للجلسات الطويلة
Bloomواضح ونظيف
Fluentمستوحى من تصميم Windows Fluent

إيجاد ثيمات إضافية

ابحث في GitHub عن spicetify theme — هناك عشرات الثيمات من المجتمع مثل Fluent وBloom وSpotify-PowerBar وغيرها.

💡 بعد كل تحديث لـ Spotify، قد تحتاج لإعادة تشغيل spicetify apply لإعادة تطبيق الثيم.
