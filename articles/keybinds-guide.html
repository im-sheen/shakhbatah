️ دليل الـ Keybinds في Hyprland

كل ما تحتاج معرفته عن اختصارات لوحة المفاتيح — من الأساسيات وحتى الـ flags والـ global keybinds وأوامر pass.

١٦ مايو ٢٠٢٦

ما هو الـ Keybind؟

الـ keybind هو تسلسل من المفاتيح تضغطها لتنفيذ أمر معين — سواء كان فتح تيرمنال، أو تشغيل برنامج، أو أي شيء آخر تختاره. في Hyprland، يتم تعريف جميع الاختصارات داخل ملف الكونفيغ.

بنية الـ Keybind

كل سطر keybind يبدأ بكلمة bind ويتبع هذا الشكل:

bind = MODIFIER, KEY, DISPATCHER, PARAMS
  • MODIFIER — مفتاح التعديل مثل SUPER أو CTRL أو ALT
  • KEY — المفتاح الأساسي مثل T أو Return
  • DISPATCHER — الأمر الذي سيُنفَّذ (الأكثر شيوعاً هو exec)
  • PARAMS — المعامل الممرر للـ dispatcher

المفاتيح المعدِّلة (Modifiers)

أشهر المعدِّلات التي ستستخدمها:

  • SUPER — مفتاح Windows أو Command
  • CTRL أو CONTROL
  • ALT

يمكنك دمج أكثر من معدِّل معاً:

bind = SUPER ALT, B, exec, notify-send "مرحباً"

الـ Dispatchers الأكثر استخداماً

exec هو الـ dispatcher الذي ستستخدمه في معظم الوقت — ينفذ أي أمر shell بشكل مباشر: 

# فتح تيرمنال
bind = SUPER, T, exec, kitty

# فتح متصفح
bind = SUPER, B, exec, brave

# إرسال إشعار
bind = SUPER ALT, B, exec, notify-send "مرحباً"

dispatchers أخرى مفيدة:

  • killactive — إغلاق النافذة الحالية
  • workspace — الانتقال لـ workspace معين
  • movetoworkspace — نقل نافذة لـ workspace آخر
  • pass — تمرير الاختصار مباشرة لتطبيق معين

الـ Bind Flags

تضيف الـ flags بعد كلمة bind مباشرة لتغيير سلوك الاختصار: 

# bind + flags
binde = , XF86AudioRaiseVolume, exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 5%+
bindl = , XF86AudioRaiseVolume, exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 5%+
Flagالمعنى
eيتكرر عند الضغط المطوّل (مثالي لمفاتيح الصوت والإضاءة)
lيعمل حتى عند قفل الشاشة
rيُنفَّذ عند الإفراج عن المفتاح لا عند الضغط
nيمرر الحدث للنافذة النشطة أيضاً (non-consuming)
💡 مثال عملي: مفاتيح الصوت والإضاءة تحتاج e للتكرار وl للعمل على شاشة القفل:
bindel = , XF86AudioRaiseVolume, exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 5%+

unbind — إلغاء اختصار

يمكنك إلغاء أي اختصار بكلمة unbind:

unbind = CTRL ALT, Delete

الاستخدام الذكي لـ unbind هو عمل toggle — سكريبت يفحص حالة ملف، فإن كان الاختصار مفعّلاً يلغيه، وإن كان ملغى يعيد ربطه.

Global Keybinds — الاختصارات العالمية

المشكلة الشائعة: اختصارات OBS لا تعمل إلا عند التركيز على نافذة OBS. الحل هو استخدام pass لتمرير الاختصار مباشرة للتطبيق: 

# تمرير Ctrl+Super+R مباشرة لـ OBS لبدء/إيقاف التسجيل
bind = CTRL SUPER, R, pass, ^(com.obsproject.Studio)$

هكذا يعمل الاختصار في أي مكان بغض النظر عن النافذة النشطة.

ربط أكثر من أمر بمفتاح واحد

يمكن تعريف نفس المفتاح أكثر من مرة لتنفيذ أوامر متعددة:

bind = SUPER, W, exec, notify-send "تم!"
bind = SUPER, W, exec, hyprctl dispatch centerwindow

ربط مفاتيح الماوس

# تحريك النوافذ بالماوس
bindm = SUPER, mouse:272, movewindow

# تغيير حجم النوافذ بالماوس
bindm = SUPER, mouse:273, resizewindow

مثال كامل — ملف binds.conf

# تطبيقات أساسية
bind = SUPER, T, exec, kitty
bind = SUPER, B, exec, brave
bind = SUPER, E, exec, thunar
bind = SUPER, R, exec, rofi -show drun

# إدارة النوافذ
bind = SUPER, Q, killactive
bind = SUPER, F, fullscreen
bind = SUPER, Space, togglefloating

# الانتقال بين الـ workspaces
bind = SUPER, 1, workspace, 1
bind = SUPER, 2, workspace, 2
bind = SUPER, 3, workspace, 3

# نقل نوافذ بين الـ workspaces
bind = SUPER SHIFT, 1, movetoworkspace, 1
bind = SUPER SHIFT, 2, movetoworkspace, 2

# صوت وإضاءة (مع تكرار + يعمل على شاشة القفل)
bindel = , XF86AudioRaiseVolume,  exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 5%+
bindel = , XF86AudioLowerVolume,  exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 5%-
bindel = , XF86MonBrightnessUp,   exec, brightnessctl set 5%+
bindel = , XF86MonBrightnessDown, exec, brightnessctl set 5%-

# ماوس
bindm = SUPER, mouse:272, movewindow
bindm = SUPER, mouse:273, resizewindow
💡 نصيحة: قسِّم ملف الكونفيغ إلى ملفات منفصلة (modularity) واجعل ملف binds.conf مستقلاً — يسهّل الصيانة كثيراً.
