Hyprlock — شاشة القفل الاحترافية لـ Hyprland

كيف تُعدّ Hyprlock من الصفر — الخلفية والإدخال والوقت والتاريخ وصورة المستخدم، مع شرح الـ config كاملاً.

١٦ مايو ٢٠٢٦

ما هو Hyprlock؟

Hyprlock هو شاشة القفل الرسمية لـ Hyprland. بخلاف حلول القفل القديمة، يدعم الـ Blur والـ Widgets والتحديث اللحظي للعناصر — مما يتيح عرض الوقت والتاريخ والأغنية الحالية وحتى حالة الطقس مباشرةً على شاشة القفل.

التثبيت

sudo pacman -S hyprlock

أو للنسخة git:

yay -S hyprlock-git

ربطه بضغطة مفتاح

في ملف إعدادات Hyprland، أضف:

bind = $mainMod, L, exec, hyprlock --config ~/.config/hypr/hyprlock/hyprlock.conf

الخيار -C أو --config يُحدد مسار ملف الإعداد. مفيد إذا كان لديك أكثر من config لثيمات مختلفة.

ملف الإعداد

Hyprlock يقرأ من:

~/.config/hypr/hyprlock.conf
📌 هذا الملف مطلوب. بدونه لن يعمل Hyprlock.

تنبيه مهم: حجم صورة الخلفية

⚠️ إذا كانت صورة الخلفية أكبر من 1 ميغابايت، سيتعطل Hyprlock. قلّل حجمها أولاً:
magick convert input.jpg -quality 85 output.jpg

قسم background — الخلفية

background {
    monitor =
    path = ~/.config/hypr/hyprlock/wallpaper

    # إعدادات الـ Blur
    blur_passes = 3
    blur_size = 7
    noise = 0.0117
    contrast = 0.8917
    brightness = 0.8172
    vibrancy = 0.1696
    vibrancy_darkness = 0.0
}

monitor إذا تركته فارغاً يطبّق الخلفية على جميع الشاشات. إعدادات الـ Blur هنا مطابقة لإعدادات الـ Blur في Hyprland — راجع مقالة إتقان Blur في Hyprland لفهمها بالتفصيل.

💡 يمكنك استخدام Symlink لجعل مسار الخلفية يتغير تلقائياً مع الثيم:
ln -sf ~/.config/hypr/themes/current/wallpaper.jpg ~/.config/hypr/hyprlock/wallpaper

قسم general — إعدادات عامة

general {
    no_fade_in = false
    grace = 0
    disable_loading_bar = false
}
  • no_fade_in — تعطيل تأثير الـ fade عند ظهور شاشة القفل
  • grace — فترة بالثواني تستطيع فيها إلغاء القفل دون إدخال كلمة مرور (0 = فوري)
  • disable_loading_bar — إخفاء شريط التحميل

قسم input-field — حقل كلمة المرور

input-field {
    monitor =
    size = 170, 45
    outline_thickness = 2
    dots_size = 0.3
    dots_spacing = 0.3
    dots_center = true
    outer_color = rgba(255,255,255,0.1)
    inner_color = rgba(0,0,0,0.5)
    font_color = rgb(255,255,255)
    fade_on_empty = false
    placeholder_text = ادخل كلمة المرور
    hide_input = false
    position = 0, -425
    halign = center
    valign = center
}

فهم نظام الإحداثيات

Hyprlock يستخدم نظام الإحداثيات الكارتيزي: المركز هو النقطة (0,0). القيم الموجبة تتجه للأعلى وللجهة اليمنى، والسالبة للأسفل وللجهة اليسرى.

  • position = 0, -425 — ينزل الحقل 425 بكسل تحت المركز
  • halign = center — محاذاة أفقية للمركز
  • valign = center — محاذاة رأسية للمركز

dots_size وdots_spacing هي نسب من ارتفاع الحقل وليست بكسلات — فكّر فيهما كـ multiplier.

outer_color وinner_color يقبلان قيم RGBA — القيمة الرابعة هي الشفافية من 0 إلى 1.

قسم label — النصوص والوقت والتاريخ

يمكنك إضافة أي عدد من labels. كل label يعرض نصاً ثابتاً أو مخرج أمر shell:

عرض الوقت والتاريخ

label {
    monitor =
    text = cmd[update:1000] echo "$(date +"%H:%M")"
    color = rgba(255,255,255,0.9)
    font_size = 72
    font_family = SF Pro Display
    position = 0, 200
    halign = center
    valign = center
}

label {
    monitor =
    text = cmd[update:1000] echo "$(date +"%A, %d %B %Y")"
    color = rgba(255,255,255,0.7)
    font_size = 22
    font_family = SF Pro Text
    position = 0, 130
    halign = center
    valign = center
}

cmd[update:1000] يعني نفّذ الأمر وحدّث النص كل 1000 ميلي ثانية (ثانية واحدة). يمكنك تغيير الرقم حسب الحاجة.

قسم image — صورة المستخدم

image {
    monitor =
    path = ~/.config/hypr/hyprlock/user-image.png
    size = 100
    border_color = rgba(255,255,255,0.3)
    border_size = 2
    rounding = -1
    position = 0, 50
    halign = center
    valign = center
}

rounding = -1 يجعل الصورة دائرية كاملة. يمكنك أيضاً استخدام Symlink هنا ليتغير أفاتار المستخدم تلقائياً مع الثيم.

إضافة Widgets ديناميكية عبر Scripts

أي label يمكنه تنفيذ سكريبت وعرض مخرجاته. مثال لعرض الأغنية الحالية من Spotify: 

# ~/.config/hypr/hyprlock/scripts/spotify.sh
#!/bin/bash
song_info=$(playerctl metadata --format "{{ title }} — {{ artist }}" 2>/dev/null)
[ -z "$song_info" ] && exit 0
echo "♫ $song_info"
label {
    monitor =
    text = cmd[update:5000] bash ~/.config/hypr/hyprlock/scripts/spotify.sh
    color = rgba(255,255,255,0.6)
    font_size = 14
    font_family = SF Pro Text
    position = 0, -500
    halign = center
    valign = center
}

ومثال لعرض حالة الطقس:

# ~/.config/hypr/hyprlock/scripts/weather.sh
#!/bin/bash
city="Baghdad"
weather=$(curl -sf "wttr.in/$city?format=%c+%t" 2>/dev/null)
[ -z "$weather" ] && echo "—" && exit 0
echo "$weather"

نصائح ختامية

  • استخدم Symlinks لربط الخلفية وصورة المستخدم بمسارات الثيمات — هكذا تتغير شاشة القفل تلقائياً عند تبديل الثيم.
  • اختبر hyprlock مباشرةً من الطرفية لتشاهد أي خطأ: hyprlock
  • إذا اعتمدت على Hypridle لتشغيل hyprlock تلقائياً، راجع مقالة إعداد Hypridle للتكامل بينهما.
  • قلّل حجم الصور دائماً إلى أقل من 1 ميغابايت قبل استخدامها كخلفية.
